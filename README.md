# Digital Aragão | Tech & Gestão - R01

Portfólio profissional desenvolvido em Flask para apresentar perfil profissional e serviços.

## Como executar

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Depois, abra no navegador:

```bash
http://127.0.0.1:5000
```

## Estrutura

- `app.py` -> aplicação Flask principal
- `templates/index.html` -> página principal
- `static/css/style.css` -> estilo visual híbrido
- `static/js/script.js` -> interações e animações
